Spacing, radius & interaction

the rules that keep it coherent
ComponentRadiusPaddingInteraction
DayColumn 14px (card) 20px 16px 24px No grid lines. Today has a 2px ink bar at top. 1px dashed divider under the date.
FixedBlockPill 12px 12px 16px Hover lifts 1px; tap opens event detail. Background = category solid.
TaskPill 12px 12px 16px 12px 18px (left pad accommodates 4px accent) Tap toggles expanded (actions inline). Swipe right = complete, swipe left = snooze (Task 10). Urgent swaps left-accent for full red border + soft red fill.
ThoughtChip 10px 6px 10px (collapsed) / 10px 12px (expanded) Dashed border by default (ghost). Hover = solid border + card fill. Tap expands in-place; “promote to task” is the primary action and is category-colored.
CaptureFAB 999px 56×56 Sticky bottom-right, 20px inset. Opens capture sheet. Scales 1.04 on hover, 0.98 on press. Focus ring is ink, not blue.
OverdueDrawer 14px 14px 0 0 16px 20px 20px Sticky under WeekView. Top border is --tier-red 1px. Collapses to just the header when empty. Count badge uses --tier-red fill.
Filter chip 10px 6px 12px Inactive = hairline border, dot = category. Active = ink fill, paper text. Horizontal scroll on mobile, scrollbar hidden.
TierFillBorderUse
1 — Fixed block --cat-[slug] solid none Calendar events + class blocks — things you can’t move.
2 — Task --surface-card (urgent: --tier-red-soft) 1px hairline + 4px left --cat-[slug]. Urgent overrides with full --tier-red. Your to-dos. Bordered because they’re mutable.
3 — Thought chip transparent 1px dashed hairline (or dotted --cat-[slug]) Surfaced Membase thoughts. Muted on purpose — don’t compete with tasks.
